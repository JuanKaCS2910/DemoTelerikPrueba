This folder contains the Telerik.Web.Spreadsheet assembly for .NET 4.0 and .NET 4.5.

It allows conversion between the JSON format used by the Kendo UI Spreadsheet
and popular file formats such as Excel (.xlsx), CSV and PDF.

This folder contains the Telerik.Web.PDF assembly for NETSTANDARD.

It allows conversion between the JSON format used by the Kendo UI PDFViewer
and PDF file.

See the documentation for more information:
http://docs.telerik.com/kendo-ui
